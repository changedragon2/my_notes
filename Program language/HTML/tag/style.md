#### Example
内部样式表
like css in html
```html
<!DOCTYPE html>
<html>
  <head>
    <style type="text/css">
      p{ color:red; }
      /* other css code here */
    </style>
  </head>
  <body>
    <p>This paragraph have red color</p>
  </body>
</html>
```

#### Attributes
| Attribute | Value       | Description                                                     |
| --------- | ----------- | --------------------------------------------------------------- |
| media     | media_query | Specifies what media/device the media resource is optimized for |
| type      | text/css    | Specifies the media type of the style tag                       |
