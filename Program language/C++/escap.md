在字符串中使用转义
```code
\\ backslash  \' single quote  \" double quote  \? question mark
\t horizontal Tab  \v vertical Tab  \b backspace
\n new line  \r carriage return
\a alter(bell) \f formfeed
# 通用转义序列
# \ followed by one, two or three octal digits
\12 newline  \7 bell  \0 null
# \x followed by one or more hexadecimal digits
\x4d M
```
