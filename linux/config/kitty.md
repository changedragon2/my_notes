GPU 加速终端，支持连字

### Configuration
复制模板
```shell
cp /usr/share/doc/kitty/kitty.conf ~/.config/kitty/kitty.conf
```
修改配置
```shell
vim ~/.config/kitty/kitty.conf

### 更改字体
font_family        FiraCode
### 字体大小
font_size          11.0
### 透明度
background-opacity    1.0
### 背景图
background_image    /path/to/png
```
